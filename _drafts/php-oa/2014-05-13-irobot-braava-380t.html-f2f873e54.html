因为 iiiday.com 的牛逼站主, 是个高生活高质量的主, 天天在别人面前推销各种东西, 然后自己又不买的人 ( 当时他本人都没海淘过, 都教了一堆人海淘 ). 然后我这人比较容易在高科技的东西面前, 没抵抗能力, 在他的忽悠下就下单拖地机器人.

原因

当然, 也是考虑到我做爸爸的身份.我家小宝宝, 以前老在地上爬和找各种东西. 所以我必须天天保证地面都是干净到可以睡在上面的. 所以每天回家都拖地 ( 标准家庭妇男, 家中女王都为你生了个宝宝, 还能让他做什么啦. ).
记住, 人生苦短, 不让人生陷入琐碎. 从此不用擦地了, 心情大好有没有.
所以在 2 月份在美国海淘的了一个用于拖地的机器人 iRobot Braava 380T ( 国内现在也有叫 Braava挚爱 ).因为一直懒, 其实早就想上来推荐这个东西.这家伙真能改变生活.

价格

这个东西比起保姆贵点, 我当时特价是买的,花了 $224.99 美元, 加 300 运费的样子, 一共大约 1700 的样子到手, 比国内 3400 的价格还是相当有优势的.大约一个月到手, 没有被税, 因为是使用的免税通道.

效果

原以为他不可能搞得很干净, 没想到还是相当不错的. 反正又不让"我的来干活". 所以我每次先用个便宜的静电纸先拖地二次 ( 建议这个地方替换成扫地机器人, 扫地, 不是拖地, 我这是个拖地机器人 ). 然后在湿拖二次. 现在 5 月了, 正常工作了三个月了. 每天这个模式, 家中很干净. 地上不在有头发和有一块一块的黑印了.
选择这种扫地和拖地的机器人, 算法和品质最为重要. 不要象国内的产品一样, 整个机器到处碰来碰去才能搞完, 这个东西有个导航, 算法不是随机的.直接是一来一回的拖地,很象日本女仆….

使用

时长: 这货可以使用连着工作二个小时, 一点问题没有.
干净程度: 湿拖一次反正达不到我的要求, 二次可以…注意, 它不是扫地的, 不要指着地上有大点的东西,它能搞它.最好配合使用.
智能程度: 以我的眼光来看, 它很傻, 虽然比起其它的来讲算法很好了. 我很希望它能和手机 app 联网.
声音: 非常安静, 不象扫地机器人, 一跑起来电视都没法看.

建议

如果其它人购买的话, 记得多买一个湿拖布回来, 虽然原生的拖布非常牛逼. 可以重复使用无数次 ( 我用了三个月的布了, 每天湿拖二次, 布还很好.). 另外,可以国内买它特有的清洁液. 这个布不要使用家中普通的清洁液容易变硬. 也不要放洗洁精之类, 机器人轮子会打滑. 国外买有快速充电器的底座, 国内没有.
